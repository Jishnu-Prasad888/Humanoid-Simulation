# Humanoid Robot Navigation in MuJoCo

## Goals

* [x] Create a 10m × 10m world
* [x] Add boundary walls

* [x] Load the humanoid model
* [ ] Add Controls to the model to make it stand upright
* [ ] Spawn it at a given `(x, y, theta)`
* [ ] Verify position and orientation are correct

* [ ] Make the humanoid move forward
* [ ] Make the humanoid turn
* [ ] Make it move to a specified `(x, y)`

* [ ] Represent the world as a grid
* [ ] Implement A* path planning
* [ ] Generate the shortest path from start to goal

* [ ] Follow the generated path
* [ ] Avoid walls and obstacles
* [ ] Reach the target `(x, y, theta)`
